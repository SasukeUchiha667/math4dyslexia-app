# Math4Dyslexia (Demo)

Қосымша: Дислексиясы бар 1–4 сынып оқушыларына арналған математикалық оқу прототипі (fake login, demo, толық функционал макеті).

## Орнату/жүгіру

1. Node.js орнатыңыз.
2. Терминалда:

```bash
git clone https://github.com/yourusername/math4dyslexia-app.git
cd math4dyslexia-app
npm install
npm start
```

3. Браузермен http://localhost:3000 деп ашыңыз.

## Ерекше мүмкіндіктер
- Фиктивті логин
- Сабақ, ойын, қатемен жұмыс, прогресс, ата-ана/мұғалім/сынып режимі
- Барлығы дислексияға, бастауышқа ыңғайлы
